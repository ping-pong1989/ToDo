Problem it solves: Helps users organize and track daily tasks in a simple console application.
Main features:
Add tasks
View tasks
Persist tasks in a TXT file
Logs all actions (add/view/save)
Who it’s for: Anyone who wants a lightweight, console-based to-do list.

"GIF"


Installation:
Clone the repository:
git clone https://github.com/yourusername/ToDo.git
Navigate to the folder:
cd ToDo

Make sure Python 3 is installed
Run the application:
python3 main.py


Usage:
dd tasks from the menu
View all tasks
Exit to save tasks permanently
All tasks are stored in tasks.txt and actions logged in app.log
